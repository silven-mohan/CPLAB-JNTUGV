# Linux Environment & Editors

## Linux Overview
Linux is a free, open-source OS kernel created by Linus Torvalds (1991). Key features:
- **Free & Open Source** - Code freely available
- **Portable** - Runs on various platforms
- **Multi-user & Multi-tasking** - Multiple users/processes simultaneously
- **Secure & Stable** - Built-in security, reliable

**Common shells**: Bash (popular), Sh, Zsh, Fish

## Vi Editor
**Visual Editor** - Lightweight, universal text editor (1976)

**Three Modes:**
1. **Command Mode** - Navigation & editing
2. **Insert Mode** - Type text (`i`, `a`, `o` keys)
3. **Ex Mode** - Advanced operations (`:` prefix)

**Quick Commands:**
- `:w` - Save | `:q` - Quit | `:wq` - Save & quit

## Vim Editor
**Vi IMproved** - Enhanced version with extra features (1991)

## Emacs Editor
**Editor MACroS** - Highly extensible, powerful editor (1976)

**Key Bindings:**
- `Ctrl+x Ctrl+f` - Open file | `Ctrl+x Ctrl+s` - Save

**Modes**: C, Python, Org, Markdown, Dired (file manager)
